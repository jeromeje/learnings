## 1. What is Template Inheritance?

**Template Inheritance** allows one HTML template to **inherit layout and structure** from another template.

- The parent template defines the **common structure**
- Child templates override only specific sections
- Avoids repeating header, footer, navbar, etc.

ðŸ“Œ It works similar to **inheritance in OOP**.

## 2. Why Template Inheritance is Important

Template inheritance helps to:
- Reduce code duplication
- Maintain consistency across pages
- Improve readability and maintenance
- Build large applications efficiently
