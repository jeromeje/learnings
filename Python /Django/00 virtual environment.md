## virtual environment: 
def:
virtual environment in Python is an isolated environment on your computer, where you can run and test your Python projects.
it allows you to manage project-specific dependencies without interfering with other projects or the original Python installation.

1. Creating a Virtual Environment:
```
python -m venv <myfirstproject>
```

2. Activate Virtual Environment :
```
myfirstproject\Scripts\activate
```
result:
(myfirstproject) file_path/ 

3. Deactivate Virtual Environment
```
deactivate
```
result: 
file_path/
