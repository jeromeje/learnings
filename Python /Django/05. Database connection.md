## 1. Database Connection in Django

Django uses **ORM (Object Relational Mapping)** to interact with databases.  
By default, Django uses **SQLite**, but it also supports **MySQL, PostgreSQL, Oracle**, etc.

4. Creating Models (Database Tables)

Models define database structure.

ğŸ”¹ Example: models.py
```
from django.db import models

class Student(models.Model):
    name = models.CharField(max_length=100)
    age = models.IntegerField()
    email = models.EmailField(unique=True)

    def __str__(self):
        return self.name
```
ğŸ“Œ Each model = One database table
ğŸ“Œ Each field = One column
