

# SECTION 3: SELECT & WHERE CLAUSE (DQL)

---

## 1. What is DQL?

**DQL (Data Query Language)** is used to **retrieve data** from database tables.

- Uses the `SELECT` statement
- Does **not modify data**
- Most frequently used SQL command
- Core of **SQL interviews**

---

## 2. SELECT Statement

### Purpose
Fetch data from one or more tables.

---

### Basic Syntax
```sql
SELECT column1, column2
FROM table_name;
```

---

Select All Columns

SELECT * FROM employee;


---

Select Specific Columns

SELECT emp_id, name, salary
FROM employee;


---

3. DISTINCT Keyword

Purpose

Removes duplicate values.

SELECT DISTINCT dept
FROM employee;

üìå Used mainly in reporting and analytics


---

4. WHERE Clause

Purpose

Filters records based on conditions.


---

Syntax

SELECT *
FROM employee
WHERE dept = 'IT';


---

Comparison Operators

Operator	Meaning

=	Equal
!= / <>	Not equal
>	Greater than
<	Less than
>=	Greater or equal
<=	Less or equal


SELECT * FROM employee
WHERE salary > 50000;


---

5. Logical Operators

AND

SELECT * FROM employee
WHERE dept = 'IT' AND salary > 50000;

OR

SELECT * FROM employee
WHERE dept = 'IT' OR dept = 'HR';

NOT

SELECT * FROM employee
WHERE NOT dept = 'Finance';


---

6. BETWEEN Operator

Purpose

Filters values within a range (inclusive).

SELECT * FROM employee
WHERE salary BETWEEN 40000 AND 60000;


---

7. IN Operator

Purpose

Matches values in a list.

SELECT * FROM employee
WHERE dept IN ('IT', 'HR', 'Admin');


---

8. LIKE Operator (Pattern Matching)

Pattern	Meaning

%	Any number of characters
_	Single character


Examples

SELECT * FROM employee
WHERE name LIKE 'A%';

SELECT * FROM employee
WHERE name LIKE '_a%';


---

9. IS NULL / IS NOT NULL

Purpose

Checks for NULL values.

SELECT * FROM employee
WHERE salary IS NULL;

SELECT * FROM employee
WHERE dept IS NOT NULL;

---

10. ORDER BY Clause

Purpose

Sorts query results.

SELECT * FROM employee
ORDER BY salary ASC;

SELECT * FROM employee
ORDER BY salary DESC;


---

11. LIMIT / OFFSET

Purpose

Restricts number of rows returned.

SELECT * FROM employee
LIMIT 5;

SELECT * FROM employee
LIMIT 5 OFFSET 10;

üìå Used heavily in pagination
